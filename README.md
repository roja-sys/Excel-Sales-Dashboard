# MIS Sales Dashboard in Excel

## Overview

This project is an interactive MIS (Management Information System) dashboard developed using Microsoft Excel.

It allows users to analyze sales performance by region using slicers, PivotTables, PivotCharts, and VBA macros.

## Features

- Interactive Region Slicer
- Top Sales Executives
- Bottom Sales Executives
- Top Target Hit %
- Bottom Target Hit %
- Bar Chart
- Pie Chart
- Line Chart
- Pivot Tables
- VBA Macros

## Tools Used

- Microsoft Excel 2021
- Pivot Tables
- Pivot Charts
- Slicers
- VBA

## Skills Demonstrated

- Data Analysis
- Dashboard Design
- Business Reporting
- Data Visualization
- Excel Automation

## Business Insights

- The dashboard identifies the top and bottom-performing sales executives.
- Regional filtering enables city-wise sales analysis.
- Target Hit % helps evaluate employee performance.
- Interactive charts provide quick visual comparisons for decision-making.
